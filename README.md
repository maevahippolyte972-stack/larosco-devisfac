# Larosco Technics — Devis & Factures

Application web (PWA) de gestion des devis et factures pour le garage **Larosco Technics**.

## Mise en ligne

Cette app est hébergée sur **GitHub Pages** :  
👉 `https://[ton-username].github.io/larosco-app/`

## Stack

- **Frontend** : HTML / CSS / JS pur — pas de build, pas de framework
- **Backend** : Supabase (PostgreSQL + Auth)
- **PDF** : jsPDF + jsPDF-autotable

## Fonctionnalités

- 🔐 Authentification par email / mot de passe
- 📊 Tableau de bord (CA mois, devis en attente, factures impayées)
- ✏️ Création de devis avec recherche de prestations en temps réel
- 💰 3 grilles tarifaires (B / C / D) — BMW = D automatique
- 🏷️ Remise libre + boutons rapides (5/10/15/20/25/30%)
- 📄 Génération PDF identique au format historique
- 📋 Liste des devis et factures
- 👥 Base clients

## Identifiants

Utilise le compte créé dans Supabase Authentication.
